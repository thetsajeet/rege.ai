# rege.ai

Rege.ai stands for Resume Generator AI

## Tech Stack

- UI: Next@15, React@19
- Stlying: Tailwindcss@4, Magicui, acternity
- Backend: FastAPI, MongoDB, Beanie

## Setup

```bash
git clone https://github.com/thetsajeet/rege.ai

# ui
cd rege.ai-ui/
pnpm install
pnpm run dev

# backend
```
